# Chapter 03: Procedures and Functions

**Learning Objectives:**
- Understand the difference between procedures and functions
- Learn how to declare and call procedures and functions
- Master parameter passing (value, var, const)
- Understand return values and how functions work
- Learn scope, visibility, and modular design principles

**SuperPascal Features Covered:**
- Procedure declarations and calls
- Function declarations, return values, and calls
- Parameter passing modes (value, var, const)
- Scope and visibility rules
- Pascal calling conventions
- Modular design principles

**Prerequisites:**
- Chapter 03 (Variables, Types, and Expressions) - Understanding variables and types

**Estimated Time:** 2-2.5 hours

**Chapter Size:** 3 H2 sections, ~20-25 pages

**Code Examples:**
- Simple procedures
- Functions with return values
- Parameter passing examples
- Scope examples

**Exercises:**
- Write procedures and functions
- Use different parameter modes
- Design reusable code
- Practice scope rules

## Notes

This chapter introduces the fundamental concept of code reuse through procedures and functions. Students should understand how to break code into reusable pieces before learning control flow.
