## Key Concepts

### Recursion Basics

- **Recursive function** - Function that calls itself
- **Base case** - Condition that stops recursion
- **Recursive case** - Part that calls itself
- **Call stack** - Stack of function calls

### When to Use Recursion

- **Natural recursion** - Problems with recursive structure
- **Tree/graph traversal** - Natural fit for recursion
- **Divide and conquer** - Break problem into subproblems
- **When simpler** - Recursive solution is clearer

### When to Use Iteration

- **Simple loops** - Iteration is clearer
- **Performance** - Iteration is often faster
- **Stack overflow** - Deep recursion can overflow
- **When simpler** - Iterative solution is clearer
