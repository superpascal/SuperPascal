# Chapter 01: Introduction to Programming on Zeal

**Learning Objectives:**
- Understand what a computer is and how it works
- Learn what programs are and how algorithms work
- Understand flow of control in programs
- Run your first SuperPascal program
- Learn to use ZealIDE for development

**SuperPascal Features Covered:**
- Basic program structure
- Platform overview (ZealZ80, Foenix65C816, FoenixA2560M)
- Development environment (ZealIDE)
- Build, run, and debug workflow

**Prerequisites:**
- Chapter 00 (Preface) - Understanding SuperPascal's philosophy

**Estimated Time:** 1.5-2 hours

**Chapter Size:** 5 H2 sections, ~40-45 pages

**Code Examples:** First program ("Hello, World!")

**Exercises:** 
- Write your first program
- Modify the example program
- Explore ZealIDE features

---

## Chapter Structure

- **01_WhatIsAComputer.md** - Computer fundamentals, CPU, memory, storage, platform overview
- **02_ProgramsAndAlgorithms.md** - What programs are, algorithms, flow of control
- **03_RunningYourFirstProgram.md** - ZealIDE usage, build/run/debug workflow, first program
- **04_ComputerArchitectureDeeperDive.md** - How code executes, memory hierarchy, CPU architecture, performance implications
- **05_Other8BitPlusSystemsThe65C816.md** - Introduction to 65C816, evolution from 6502, banked memory overview

---

## Learning Path

- **Before:** Chapter 00 (Preface)
- **After:** Chapter 03 (Variables, Types, and Expressions)

---

## Notes

This chapter establishes the foundation. Students should understand computers at a basic level before learning to program. The first program should be simple but complete.
