# Intrinsics Overview

**Part of:** [06_IntrinsicsAndDirectives.md](../06_IntrinsicsAndDirectives.md)

---

Intrinsics are **built-in procedures and functions** that map directly to hardware operations or runtime services. They provide efficient access to Zeal platform features.

**Properties:**
- Cannot be redefined
- Type-checked by compiler
- Generate optimized code sequences
- Platform-specific (Zeal 8-bit Computer)
